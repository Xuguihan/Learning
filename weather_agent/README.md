\#天气助手项目存档



> \*\*项目名称\*\*：天气助手  

> \*\*创建平台\*\*：\[Dify]  

> \*\*用途\*\*：用于练习工程化部署与天气问答



\---



\### 项目结构



本仓库保存了 Dify 平台上的机器人完整配置文件，结构如下：



weather\_agent/

├── weather\_agent.yml        # Dify 导出的 Agent 配置文件

└── README.md                   # 项目说明文档



\### 如何使用（还原机器人）

> 1. 登录 Dify 控制台

> 2. 点击“导入 DSL”

> 3. 选择 `weather\_agent.yml`

> 4. 完成！
