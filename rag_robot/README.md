# RAG 机器人项目存档


**项目名称**：RAG 机器人  

**创建平台**：[Dify] 

**用途**：用于练习工程化部署与知识库问答


### 项目结构

本仓库保存了 Dify 平台上的机器人完整配置文件，结构如下：
Learning/
└── rag\_robot/
├── rag\_robot\_config.yml    # \[必看] Dify 导出的机器人配置文件
└── README.md               # 你正在看的说明文档

### 如何使用（还原机器人）

1. 登录 Dify 控制台

2. 点击“导入 DSL”

3. 选择 `rag_robot.yml`

4. 完成！
